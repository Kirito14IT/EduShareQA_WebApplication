# EduShareQA 功能测试指南

## 测试账号

系统提供了三个测试账号，分别对应不同的角色：

| 角色 | 用户名 | 密码 | 说明 |
|------|--------|------|------|
| 学生 | `student01` | `password123` | 可以浏览资源、提问、上传资源 |
| 教师 | `teacher01` | `password123` | 可以回答问题、发布资源、查看学生提问 |
| 管理员 | `admin01` | `password123` | 可以管理课程、教师、资源和问答内容 |

## 测试流程

### 1. 学生功能测试

#### 1.1 登录测试
1. 访问登录页面
2. 使用 `student01` / `password123` 登录
3. 验证成功跳转到资源列表页面

#### 1.2 资源浏览
1. 在"学习资源"页面，测试：
   - 按课程筛选（选择"线性代数"、"大学英语"等）
   - 关键字搜索（搜索"线性代数"、"期末"等）
   - 分页功能（如果数据超过8条）
   - 点击资源标题查看详情

#### 1.3 资源上传
1. 点击"资源上传"
2. 填写表单：
   - 标题：测试资源
   - 简介：这是一个测试资源
   - 所属课程：选择任意课程
   - 上传文件（可选）
3. 提交后验证资源出现在列表中

#### 1.4 提问功能
1. 点击"我要提问"
2. 填写表单：
   - 课程：选择"线性代数"
   - 标题：测试问题
   - 内容：这是一个测试问题
   - 上传图片附件（可选）
3. 提交后验证问题出现在"我的提问"列表中

#### 1.5 提问搜索
1. 点击"提问搜索"
2. 测试：
   - 按学科搜索（选择"线性代数"）
   - 按教师搜索（选择"Bob Teacher"）
   - 关键字搜索（搜索"矩阵"）
   - 查看搜索结果

#### 1.6 个人中心
1. 点击"个人中心"
2. 在"我的资源"标签页：
   - 查看自己上传的资源
   - 点击"编辑"修改资源信息
   - 点击"删除"删除资源
3. 在"我的提问"标签页：
   - 查看自己提出的问题
   - 点击"查看"查看问题详情和回答
   - 点击"编辑"修改问题（仅限待回答状态）
   - 点击"删除"删除问题

#### 1.7 通知功能
1. 等待教师回答问题后，查看通知按钮是否显示"您有 X 条新回答"
2. 点击通知按钮进入通知中心
3. 查看通知统计信息

### 2. 教师功能测试

#### 2.1 登录测试
1. 退出当前账号
2. 使用 `teacher01` / `password123` 登录
3. 验证导航菜单显示教师专用菜单

#### 2.2 工作台
1. 进入"工作台"页面
2. 查看统计卡片：
   - 待回答问题数
   - 已发布资源数
   - 已回答问题数
3. 点击卡片跳转到对应页面

#### 2.3 学生提问
1. 进入"学生提问"页面
2. 查看所属课程的学生提问列表
3. 测试状态筛选（待回答、已回答、已关闭）
4. 点击"查看/回答"进入问题详情页

#### 2.4 回答问题
1. 在问题详情页，查看问题内容
2. 在回答表单中：
   - 输入回答内容
   - 上传附件（可选）
3. 提交回答
4. 验证回答出现在问题详情中

#### 2.5 发布资源
1. 点击"发布资源"
2. 填写资源信息并上传文件
3. 提交后验证资源出现在资源列表中

### 3. 管理员功能测试

#### 3.1 登录测试
1. 退出当前账号
2. 使用 `admin01` / `password123` 登录
3. 验证导航菜单显示管理员专用菜单

#### 3.2 课程管理
1. 进入"课程管理"页面
2. 测试"新建课程"：
   - 填写课程编号：TEST101
   - 填写课程名称：测试课程
   - 填写开课学院：测试学院
   - 填写课程描述（可选）
   - 提交后验证课程出现在列表中
3. 测试"编辑课程"：
   - 点击某个课程的"编辑"按钮
   - 修改课程信息
   - 提交后验证更新成功
4. 测试"删除课程"：
   - 点击某个课程的"删除"按钮
   - 确认删除
   - 验证课程从列表中消失

#### 3.3 教师管理
1. 进入"教师管理"页面
2. 测试"新建教师"：
   - 填写教师信息（用户名、邮箱、姓名等）
   - 填写职称、所属学院、简介（可选）
   - 提交后验证教师出现在列表中
3. 测试"编辑教师"：
   - 点击某个教师的"编辑"按钮
   - 修改教师信息
   - 提交后验证更新成功
4. 测试"分配课程"：
   - 点击某个教师的"分配课程"按钮（书本图标）
   - 在弹窗中选择要分配的课程（可多选）
   - 提交后验证课程分配成功
5. 测试"删除教师"：
   - 点击某个教师的"删除"按钮
   - 确认删除
   - 验证教师从列表中消失

#### 3.4 资源管理
1. 进入"资源管理"页面
2. 查看所有资源列表
3. 测试关键字搜索
4. 测试删除资源：
   - 点击某个资源的"删除"按钮
   - 确认删除
   - 验证资源从列表中消失

#### 3.5 问答管理
1. 进入"问答管理"页面
2. 查看所有问题列表
3. 测试删除问题：
   - 点击某个问题的"删除"按钮
   - 确认删除
   - 验证问题从列表中消失

### 4. 通用功能测试

#### 4.1 个人设置
1. 使用任意账号登录
2. 点击"个人设置"
3. 在"个人资料"标签页：
   - 修改姓名、邮箱、所属学院
   - 提交后验证更新成功
4. 在"修改密码"标签页：
   - 输入旧密码
   - 输入新密码（至少6位）
   - 提交后验证密码修改成功

**资源：**

**问题：**
- 矩阵对角化条件（已回答）
- 概率统计复习范围（待回答）

### 注意事项

1. **数据持久化**：当前使用模拟数据，刷新页面后数据会重置
2. **文件上传**：文件上传功能为模拟，实际文件不会保存
3. **权限控制**：不同角色看到的菜单和功能不同
4. **通知功能**：通知数量基于问题状态计算，当问题被回答后，通知数量会增加

## 已知限制

1. 文件上传功能为模拟，实际文件不会保存到服务器
2. 数据存储在内存中，刷新页面后会重置
3. 教师分配课程功能中，课程列表需要先创建课程才能分配
4. 通知功能基于问题状态，需要教师回答问题后才会触发

## 测试建议

1. **按角色测试**：分别使用三个不同角色的账号进行完整测试
2. **功能完整性**：确保每个功能模块的增删改查都能正常工作
3. **交互体验**：注意页面动画和交互效果是否流畅
4. **错误处理**：测试边界情况，如空数据、删除确认等
5. **权限验证**：验证不同角色只能访问对应的功能
